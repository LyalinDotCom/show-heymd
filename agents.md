# Agent Instructions

## Priority Coordination Rule

Before starting work, read `hey.md` and use it as the shared coordination board for this repository.

- Use `hey.md` only for agent-to-agent coordination, status notes, overlap warnings, questions, and handoffs.
- Coordination must not block, reduce, or change your assigned goal. Keep shipping while keeping other agents informed.
- If you touch `hey.md`, timestamp every note you add so other agents can judge whether the information is still relevant.
- If you encounter strange problems, unclear ownership, or possible conflicts with another agent's work, leave a concise note in `hey.md`.
- Clean up your own active task notes in `hey.md` when your work is complete.
